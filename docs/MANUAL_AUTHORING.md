# Manual Authoring

Manual authoring is available in `Admin > Content Import` by selecting a blueprint mapping and clicking `Add Manual MCQ`, `Add Manual Flashcard`, or `Add Manual Vignette`.

## Required Mapping

Manual items must be mapped to:

- Exam track
- Topic/domain
- Optional competency/subtopic
- Optional Social Work applied knowledge statement
- Difficulty
- Cognitive level

The selected mapping is applied to every preview item at import time. This acts as bulk metadata editing for a group of imported or manually authored drafts.

## MCQ Fields

Manual MCQs include:

- Question
- Options A-D
- Correct answer
- Rationale

Imported MCQs are inactive, unreviewed drafts. Use `Run integrity review` during import or run integrity from the question review page.

## Flashcard Fields

Manual flashcards include:

- Front
- Back

They are stored as inactive, unreviewed drafts.

## Case Vignette Fields

Manual case vignettes include:

- Case
- Prompt
- Ideal answer
- Coaching feedback

They are stored as inactive, unreviewed drafts.

## Publishing

Manual and imported content follows the same publication rules as generated content. MCQs must pass integrity gates or have an admin override with a reason, admin ID, and timestamp. Flashcards and vignettes must be reviewed and activated by an admin before students can retrieve them.
